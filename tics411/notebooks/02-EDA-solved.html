# Mostrar cómo se construye esta función...
def calculate_quantiles(column):
    quantiles = iris_df.quantile([0.25, 0.75], numeric_only=True)
    iqr_sl = quantiles.loc[0.75, column] - quantiles.loc[0.25, column]
    return iqr_sl

Cuando hacemos visualizaciones tenemos 2 formas de hacerlas: * Usar Pandas directamente y/o Matplotlib: Buena documentación, gráficos bonitos. * Usar Seaborn: Documentación Complicada, fácil para descubrir insights.
